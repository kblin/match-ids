# Match IDs

Match IDs (or more precisely `locus_tag`s) between two GenBank files.

## Installation

Run inside a Python virtual environment

```bash
pip install -r requirements.txt

```

## Usage

```
./match_ids.py reference.gbk query.gbk > matches.tsv
```
